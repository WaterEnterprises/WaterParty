The Product: WaterParty

WaterParty is a Tinder-style social discovery application that unifies the entire party-going journey:
• Swipe to discover parties, events, and gatherings nearby
• RSVP and see which friends are attending
• Real-time chat with hosts and attendees
• Integrated payments — tipping, crowdfunding, and cover charges via Stripe Connect
• Automatic currency detection via GPS for cross-border functionality
• Reputation system for hosts and attendees

Technology Stack (fully functional MVP):
• Front-end: React 19 + TypeScript + Vite
• Back-end: Bun runtime + Turso (edge SQLite)
• Payments: Stripe Connect
• Real-time: WebSockets
• Cross-platform: iOS + Android (Capacitor)
• GPS-based currency detection
