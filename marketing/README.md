## 🎯 Key Selling Points

> Ready-made bullets for any pitch or proposal:

- **First-mover** in Tinder-style party discovery
- **Cross-platform** — iOS + Android from a single codebase (React 19 + Capacitor)
- **Monetized from day one** — Stripe-powered tipping (5%), crowdfunding (5%), AdMob ads
- **Real-time** — WebSocket chat, live feed updates, push notifications
- **Scalable** — Edge-hosted SQLite (Turso), Bun runtime, ~50ms response times
- **GPS-native** — Location-based discovery with automatic currency detection (BRL, USD, EUR, GBP)
- **Global by design** — Multi-currency engine works across borders without configuration
- **Umbrella strength** — Part of Water Enterprises / Stellarium Foundation ecosystem (alongside Water AI, robotics, decentralized compute)

## ✍️ How to Use

1. **Pitch to investors** — Start with `executive-summary.md`, then use `pitch-deck-outline.md` for presentations
2. **Pitch to sponsors** — Use `sponsorship-deck.md` (English) or `sponsorship-deck-pt.md` (Portuguese)
3. **Outreach** — Use `contact-list.md` for the target list and email templates
4. **Track progress** — Create a CRM spreadsheet using the template in `contact-list.md`
5. **Update quarterly** — Refresh `financial-projections.md` as real data accumulates
6. **Keep version-controlled** — All materials tracked in git alongside the codebase
